# Hanabi Canvas

A collaborative pixel art toy where players draw 32x32 pixel art, launch it as a firework, and watch it explode into particles that crystallize back into their drawing.

Built for game jam events.

## Features

- **Drawing Canvas** -- 32x32 pixel grid with 8-color palette, draw/erase tools, undo/redo
- **Firework Launch** -- Pixel art rockets into the sky and bursts into a particle display
- **Crystallization** -- Chaos particles steer back into the original drawing formation
- **Free Mode** -- Draw anything and launch
- **Challenge Mode** -- Color constraints and themed prompts from a request board
- **Gallery** -- Slideshow of past creations with a like system
- **Export/Import** -- JSON format for sharing data between sessions

## Tech Stack

- Unity 2023.2.x
- Universal Render Pipeline (URP)
- ScriptableObject data-driven architecture
- Custom particle system (procedural mesh)
- Standalone target
